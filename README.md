# toby-sandbox

Test playground for [toby](https://github.com/0xtiby/toby). A simple CLI task tracker meant to be built entirely by toby from specs.

This project contains only specs and a reset script — all code is generated by toby.

## Usage

```bash
# Init toby in this project
toby init

# Run toby on specs
toby plan --all
toby build --all

# Reset to clean state
./reset.sh
```
